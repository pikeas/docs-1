    ExUnit v1.0.5 ExUnit.Test

    A struct that keeps information about the test.

    It is received by formatters and contains the following fields:

    • :name - the test name
    • :case - the test case
    • :state - the test state (see ExUnit.state)
    • :time - the time to run the test
    • :tags - the test tags
